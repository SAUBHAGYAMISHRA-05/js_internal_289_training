# C# Delegates Assignment - Day 6

## Project 1: Smart Online Shopping Discount System

### Features:
- Product, Customer, and ShoppingCart classes
- CustomerType enum (Occasional/Regular)
- DiscountStrategy delegate
- Dynamic discount application based on customer type
- Festival Discount: 17% for Occasional customers
- Premium Discount: 28% for Regular customers

### Key Concepts Demonstrated:
- Delegate definition and usage
- Strategy Pattern implementation
- Encapsulation of discount logic
- Runtime method binding
