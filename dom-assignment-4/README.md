# DOM & Events - Assignment 4 (Day 4)

A complete interactive web application demonstrating JavaScript DOM manipulation and event handling.

## Features Implemented:

### 12 Interactive Tasks:

1. **Change Text on Button Click** - Modify paragraph content dynamically
2. **Background Color Changer** - Red, Blue, Green, Reset buttons
3. **Image Switcher** - Next/Previous buttons to cycle through images
4. **Show/Hide Paragraph** - Toggle paragraph visibility
5. **Counter App** - Increase/Decrease/Reset counter with zero limit
6. **Create Element Dynamically** - Add colored boxes with hover effects
7. **Form Input Reader** - Capture and display user input
8. **Todo List** - Add and delete tasks dynamically
9. **Attribute Change** - Resize images using setAttribute()
10. **Theme Switcher** - Toggle between light/dark modes
11. **Live Clock** - Real-time updating clock (Bonus)
12. **Random Quote Generator** - Display random quotes (Bonus)

## How to Use:

1. Clone or download the project
2. Open `index.html` in any modern web browser
3. Interact with all 12 tasks in the interface
